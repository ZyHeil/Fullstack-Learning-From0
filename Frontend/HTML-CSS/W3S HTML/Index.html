HTML CSS

CSS stands for Cascading Style Sheets.
CSS saves a lot of work. It can control the layout of multiple web pages all at once.

U can use CSS by:

  • Inline - by using the style attribute inside HTML elements

    • This is a Blue Border using inline css


  • Internal - by using a style element in the head section

    • An internal CSS is used to define a style for a single HTML page.

  • External - by using a < link > element to link to an external CSS file

    • Using a stylesheet css file. Note that this will override any style that is used with the internal method

Some property like:

  • color
  • font-family
  • font-size
  • Border
  • padding or space between text and border
  • Margin or space outside of the border

        The target attribute specifies where to open the linked document.

        The target attribute can have one of the following values:

        _self - Default. Opens the document in the same window/tab 
            as it was clicked

        _blank - Opens the document in a new window or tab

        _parent - Opens the document in the parent frame

        _top - Opens the document in the full body of the window
